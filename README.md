# PRODIGY_DS_02

## Titanic Dataset - Data Cleaning & Exploratory Data Analysis (EDA)

This project is a part of my **Prodigy InfoTech Data Science Internship (Task 02)**. The objective is to perform data cleaning and exploratory data analysis (EDA) on the Titanic dataset to understand the relationships between different features and identify factors that influenced passenger survival.

### Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

### Highlights

* Handled missing values and cleaned the dataset.
* Explored survival patterns based on gender, age, and passenger class.
* Created visualizations to identify trends and relationships.
* Performed correlation analysis to gain meaningful insights.

**Dataset:** Titanic Dataset (Kaggle)
